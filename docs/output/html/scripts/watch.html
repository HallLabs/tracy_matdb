Context Execution Watcher¶

Depending on the matdb context, it is often useful to watch the output that is being piped to the batch output file. This script is a simple handler around tail -f that keeps track of file paths for output files.

matdb.scripts.matdb_watch.examples()[source]¶

Prints examples of using the script to the console using colored output.

matdb.scripts.matdb_watch.run(args)[source]¶

Runs the matdb setup and cleanup to produce database files.
