Tracy Queue submission¶

Once a database of configurations has been built and prepped for submission to the Tracy Queue this script will grab the data and submit it to the job script.

matdb.scripts.tracy_sub.examples()[source]¶

Prints examples of using the script to the console using colored output.

matdb.scripts.tracy_sub.run(args)[source]¶

Runs the matdb setup and cleanup to produce database files.

matdb.scripts.tracy_sub.submit(subfil, test=False)[source]¶

Submits the file to the queue if test is True.

Parameters:
  • subfil (str) – the json file to be loaded.
  • test (bool) – True if a test run.
