Database Converter¶

Because matdb supports many different interatomic potential trainers, and since each possibly has its own custom format for the configuration databases, conversion is inevitable. Matdb uses HDF5 for compactness, but can convert between formats using this script.

matdb.scripts.matdb_convert.examples()[source]¶

Prints examples of using the script to the console using colored output.

matdb.scripts.matdb_convert.run(args)[source]¶

Runs the matdb setup and cleanup to produce database files.
