Supercell Selector¶

When producing hessians with the Hessian group, a supercell needs to be selected. This script streamlines the selection process for multiple seeds and sizes.

matdb.scripts.matdb_supercell.examples()[source]¶

Prints examples of using the script to the console using colored output.

matdb.scripts.matdb_supercell.run(args)[source]¶

Runs the matdb setup and cleanup to produce database files.
