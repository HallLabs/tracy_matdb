Special Paths in k-Space¶

To automatically produce the phonon band structure plots, we need to have an appropriate special path in the BZ. matdb interfaces with materialscloud.org to extract their recommended path in the BZ.

matdb.kpoints.custom(target, key, atoms=None)[source]¶

Creates the KPOINT file with the specified custom configuration in target.

Parameters:
  • target (str) – path to the directory to create the KPOINTS file in.
  • key (str) – one of [‘gamma’], specifies which custom KPOINTS file to generate (see note above).
  • atoms (matdb.atoms.Atoms) – atoms object to generate KPOINTS for.
matdb.kpoints.find_qpoints(atoms, supercell)[source]¶

Finds the q points that would be “pinned down” by a calculation of the given supercell.

Parameters:
  • atoms (matdb.atoms.Atoms) – primitive atoms object.
  • supercell (numpy.ndarray) – array of shape (3, 3) or a list/tuple of shape 3 or 9.
matdb.kpoints.kpath(atoms)[source]¶

Returns a list of the special k-points in the BZ at which the phonons should be sampled.

Parameters:atoms (matdb.atoms.Atoms) – structure to get the path for.
Returns:(path, points), where path is a list of names of the special points and points is a dict with the same names as keys and numpy.ndarray as values.
Return type:tuple
matdb.kpoints.parsed_kpath(atoms)[source]¶

Gets the special path in the BZ for the structure with the specified atoms object and then parses the results into the format required by the package machinery. :param atoms: a matdb Atoms object. :type atoms: matdb.atoms.Atoms

Returns:result of querying the materialscloud.org special path service. First term is a list of special point labels; second is the list of points corresponding to those labels.
Return type:tuple
