Logger Module¶

Implements logging for managing statistics and debugging for matdb databases, groups, fitters, analyzers, etc.

class matdb.logs.Logger(root, identifier)[source]¶

Handles the logging of events for a matdb.

Parameters:
  • root (str) – path to the root logs directory for this logger.
  • identifier (str) – name of the log (unique in the application).
