Manual Database Documentation¶

To use this database the user must place every desired configuration of atoms they want in the database into the SEED directory of the root directory specified in the matdb.yml file.

Example yml File Entry¶

steps:
- type: 'simple.Manual'
  seeds: ["POSCAR:PdAg25","POSCAR:PdAgp0","POSCAR:PdAg75"]

Manual Database¶

A database of nothing but seed configurations.

class matdb.database.simple.Manual(name='manual', extractable=True, **dbargs)[source]¶

A basic group that just sets up a calculator for each atoms object specified by the user.

Parameters:
  • name (str) – the name of the database group.
  • extractable (bool) – True if a calculation is to be performed.
  • dbargs (dict) – a dictionary of arguments for the Group class.

Note

Additional attributes are also exposed by the super class Group.

name¶

Name of this database type relative to the database collection. This is also the name of the folder in which all of its calculations will be performed.

Type:str
can_extract()[source]¶

Runs post-execution routines to clean-up the calculations. This super class implementation only checks that each of the sub-config directories has the necessary files needed for extraction of output.

fitting_configs¶

Returns a AtomsList for all configs in this group.
