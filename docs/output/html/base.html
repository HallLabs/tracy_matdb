Abstract Functions¶

class matdb.base.abstractstatic(function)[source]¶
matdb.base.debug = False¶

when True, loggers throughout the matdb system are set to debug level to produce more verbose logging output. Can also be int corresponding to level in logging.

Type:bool
matdb.base.deprecated(func)[source]¶

This is a decorator which can be used to mark functions as deprecated. It will result in a warning being emitted when the function is used.

matdb.base.exhandler(function, parser)[source]¶

If “examples” was specified in ‘args’, the specified function is called and the application exits.

Parameters:
  • function – the function that prints the examples.
  • parser – the initialized instance of the parser that has the additional, script-specific parameters.
matdb.base.set_debug(debugging)[source]¶

Sets the package debug mode.

matdb.base.set_testmode(testing)[source]¶

Sets the package testing mode.

matdb.base.testmode = False¶

when True, the package is operating in unit test mode, which changes how plotting is handled.
