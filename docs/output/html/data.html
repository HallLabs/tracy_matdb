Elemental and Alloy Systems Data¶

Data about elemental and alloy systems that doesn’t change with time.

matdb.data.vegard(elements, concs)[source]¶

Interpolates the lattice parameters for the given alloy system using vegard’s law.

Parameters:
  • elements (list) – list of str chemical symbols; keys in latpars.
  • concs (list) – list of concentrations for each element in elements.
