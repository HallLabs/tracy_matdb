Messaging Module¶

This module handles writing to the terminal or a log file with support for coloring for warnings, errors, etc.

matdb.msg.arb(text, cols, split)[source]¶

Prints a line of text in arbitrary colors specified by the numeric values contained in msg.cenum dictionary.

matdb.msg.blank(n=1, level=2)[source]¶

Prints a blank line to the terminal.

matdb.msg.cenum = {'cerrs': 0, 'cgens': 3, 'cinfo': 2, 'cokay': 5, 'cstds': 4, 'cwarn': 1}¶

Dict of the various colors available for coloring specific lines in the arb().

matdb.msg.err(msg, level=-1, prefix=True)[source]¶

Prints the specified message as an error; prepends “ERROR” to the message.

matdb.msg.example(script, explain, contents, requirements, output, outputfmt, details)[source]¶

Prints the example help for the script.

matdb.msg.gen(msg, level=1)[source]¶

Prints the message as generic output to terminal.

matdb.msg.info(msg, level=1)[source]¶

Prints the specified message as information.

matdb.msg.nocolor = False¶

When true, the colored outputs all use the regular print().

matdb.msg.okay(msg, level=1)[source]¶

Prints the specified message as textual progress update.

matdb.msg.printer(text, color=None, **kwargs)[source]¶

Prints using color or standard print() depending on the value of ‘nocolor’.

matdb.msg.quiet = None¶

When in quiet mode, no text is ever printed to terminal unless its verbosity level is explicitly specified.

matdb.msg.set_quiet(is_quiet)[source]¶

Sets whether the messaging system is running in quiet mode. Quiet mode only prints messages with explicit verbosity specified. If verbosity==None and quiet==True, any message with level >= 0 is printed.

matdb.msg.set_verbosity(level)[source]¶

Sets the modules message verbosity level for all messages printed.

Parameters:level – a positive integer (>0); higher levels including more detail.
matdb.msg.std(msg, level=1)[source]¶

Prints using the standard print() function.

matdb.msg.verbosity = None¶

The verbosity level of messages being printed by the module.

matdb.msg.warn(msg, level=0, prefix=True)[source]¶

Prints the specified message as a warning; prepends “WARNING” to the message.

matdb.msg.will_print(level=1)[source]¶

Returns True if the current global status of messaging would print a message using any of the printing functions in this module.
